Предприятие: рекламное агентство.

ОПИСАНИЕ ПРОГРАММЫ.
Данная программа предназначена для составления клиентских заказов.
В данной программе предусмотрено:
1.1
	Создание новых заказов;
	Изменение данных о заказе;
	Удаление заказа;
	Поиск заказа по идентификатору;

1.2 
	Создание новых товаров (рекламной продукции);
	Изменение данных о товарах;
	Удаление товаров;
	Поиск по идентификатору товара;

Правила выполнения программы:
1. Если статус клиента выполненый то заказ автоматически удаляется с его клиентом
и сотрудником;
2. В программе не может повторяться номер заказа.

ФУНКЦИИ ПРОГРАММЫ:
1. Вывод в консоль информацию о товарах;
2. Вывод в консоль подробной информации о заказах;
3. Добавление, редактирование, удаление, получение товара по идентификатору;
4. Добавление, редактирование, удаление, получение заказа по идентификатору. 
	
ПРЕДМЕТНАЯ ОБЛАСТЬ.
	
Сущности предметной области данной темы являются:
1) Клиент - тот, кто делает заказ на рекламную продукцию;
2) Сотрудник - тот, кто отслеживает выполнение работы и выполняет заказ;
3) Товар - это рекламная продукция, которую предлагает предприятие;
4) Заказ - это список товаров, который готов купить клиент;
